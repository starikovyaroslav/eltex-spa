# Eltex SPA

Тестовое задание для компании Eltex

### Что использовалось:
* Angular 15
* TypeScipt
* Material Angular
* RxJS

### Запуск проекта 
1. Клонировать репозиторий
2. Установить зависимости 
```sh
npm install
```
3. Запустить проект
```sh
npm start
```
5. Проект запускается на http://localhost:4200/

## Посмотреть можно здесь: https://eltex-spa.vercel.app/
